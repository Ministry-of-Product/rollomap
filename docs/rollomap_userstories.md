# RolloMap User Stories

## Founder / Operator

- As a Founder I want to identify people interested in my idea so that I can reach out effectively because I need early traction
- As a Founder I want a briefing before meetings so that I can show up prepared because relationships matter

## Professional

- As a Professional I want to recall past conversations so that I can continue them naturally because I value relationships
- As a Professional I want to find connections in a domain so that I can leverage my network because opportunities depend on people

## Network Builder

- As a Network Builder I want to track interactions so that I can maintain relationships because consistency builds trust
- As a Network Builder I want reminders of neglected contacts so that I can reconnect because relationships decay over time
